# Image Generation Prompts

Master list of prompts for generating images.

## Prompt Structure

```
[subject], [action/pose], [clothing], [setting], [lighting], [camera], [style], [quality]
```

## Quality Tags

### Always Include
```
high quality, detailed, sharp focus, professional photography
```

### For Realism
```
photorealistic, raw photo, 8k uhd, dslr, film grain
```

### For Stylized
```
digital art, trending on artstation, highly detailed illustration
```

## Subject Description

Base description for the character (update this once finalized):
```
[Your character description here]
```

## Pose Library

### Standing Poses
- standing confidently, hands on hips
- casual standing pose, relaxed
- leaning against wall, crossed arms

### Sitting Poses
- sitting on chair, legs crossed
- sitting casually on couch
- sitting on floor, relaxed pose

### Action Poses
- walking, mid-stride
- laughing, candid moment
- looking over shoulder

## Setting/Background Options

### Indoor
- modern apartment, minimalist interior
- cozy bedroom, soft lighting
- professional studio, clean background
- gym interior, workout equipment

### Outdoor
- city street, urban background
- beach, ocean in background
- park, natural greenery
- rooftop, city skyline

## Lighting Options

- natural window light, soft shadows
- golden hour, warm sunlight
- studio lighting, professional
- neon lights, cyberpunk aesthetic
- ring light, beauty lighting

## Camera/Shot Types

- portrait shot, upper body
- full body shot
- close-up face
- medium shot, waist up
- three-quarter view

## Style Modifiers

- instagram aesthetic
- editorial photography
- candid street photography
- professional headshot
- lifestyle photography
